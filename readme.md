Desafio Backend Frameworks

Esse projeto foi feito pra comparar Node.js com Express e Java com Spring Boot, focando na organização em camadas (MVC).

O repositório tem duas pastas, uma pra cada tecnologia. Em ambas eu criei as camadas de model, controller e routes/service, só pra representar a estrutura.

O Express é mais simples e rapido de usar, com menos código. Já o Spring Boot é mais organizado, mas mais verboso e complexo no inicio.

O Express foi criado em um ambiente online devido as limitações do computador, mas foi feito certinho, ja o Spring Boot foi a estrutura atraves da Spring initializr.

A ideia principal foi entender como as camadas se comunicam e organizar melhor o código, mesmo sem implementar tudo completo.
